# Home Page Hero Images

Put homepage hero images in this folder.

Expected filenames (so you can switch them via `/?hero=1`, `/?hero=2`, `/?hero=3`):

- `coffee-cherries-1.jpg`
- `coffee-cherries-2.jpg`
- `coffee-cherries-3.jpg`

Notes:

- Use JPG (recommended) or PNG.
- Prefer wide landscape images.
- Keep file sizes reasonable for the web (ideally < ~500KB each).
